Features

Intelligent photo management,
beautifully simple.

AI Duplicate Detection

Combines Apple Vision feature prints with perceptual hashing to find exact and near-duplicate photos with exceptional accuracy.

Smart Quality Assessment

Multi-signal scoring system evaluates sharpness, exposure, composition, and aesthetics. Each photo receives a clear A-F quality grade.

Intelligent Grouping

DBSCAN clustering groups visually similar photos together, making it effortless to choose the best shot from any burst or series.

How It Works

Five-phase intelligence pipeline.

01

Import

Select your photo library or drag folders. Snap Sieve reads metadata and thumbnails instantly.

02

Analyze

Feature prints and perceptual hashes are computed on-device using Apple Vision and Core ML.

03

Compare

Cosine similarity and Hamming distance find duplicates and near-matches across your entire library.

04

Cluster

DBSCAN groups similar photos into visual clusters, surfacing the best candidates from each set.

05

Review

AI suggests which to keep, but you make every decision. Move to Trash only when you choose.

Under the Hood

Three ML components, one seamless experience.

Feature Prints
Generates high-dimensional vector embeddings via Apple Vision's VNFeaturePrintObservation for semantic image comparison.
Powers duplicate detection and similarity clustering with cosine distance calculations.
Aesthetics Model
Core ML model evaluating composition, exposure, sharpness, and visual appeal to produce a multi-signal quality score.
Assigns A-F grades so you can instantly identify your best and worst shots.
Perceptual Hash
Generates compact binary fingerprints resilient to resizing, compression, and minor edits using DCT-based hashing.
Catches exact and near-exact duplicates that feature prints alone might score differently.
